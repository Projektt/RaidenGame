
MOVEMENT:

W: Up
A: Left
S: Down
D: Right

QUIT GAME:

ESC: To quit the game. 

SHOOT:

J: Shoot

HEALTH:

Top left corner has 5 HP Bar before plane dies. Which will end the game. 
There is only 1 life before having to restart the round. 

Resolution:

Game is default 1280 by 720 to be a small browser game not full screen. 

Number of Enemy Ships:

Number of enemy ship per ship can be adjusted on settings.
